# Articles

Reader-facing articles for `eva-theory`.

This directory holds public-facing writing that explains EVA at different depths and for different audiences. These files do not replace the main theory document in `THEORY/`; they provide additional entry points into it.

## Files in this directory

- `01-paradigm-introduction.md` — introductory article on the existence-centered paradigm and how it differs from task-centered agent design
- `02-architectural-contributions.md` — companion article focused on EVA's four main architectural distinctions
- `03-related-work-and-positioning.md` — positioning note relating EVA to adjacent agent paradigms, cognitive traditions, and safety/control work

## Suggested reading order inside ARTICLES

1. `01-paradigm-introduction.md`
2. `02-architectural-contributions.md`
3. `03-related-work-and-positioning.md`

## Directory boundary

`ARTICLES/` is for reader-facing explanatory writing. It is not the authoritative source for the full theory text, implementation status, or roadmap details.

- For the full theory, see `THEORY/v0.5-integrated.md`
- For visuals, see `VISUALS/README.md`
- For the theory/implementation boundary note, see `IMPLEMENTATION/eva-agent-correspondence.md`
