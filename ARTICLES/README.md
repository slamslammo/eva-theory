# Articles

Reader-facing articles for `eva-theory`.

This directory holds public-facing writing that explains EVA at different depths and for different audiences. These files do not replace the main theory document in `THEORY/`; they provide additional entry points into it.

## Current public-facing articles

- `01-paradigm-introduction.md` — reader-oriented introduction to the existence-centered paradigm and how it differs from task-centered agent design
- `02-architectural-contributions.md` — engineering-oriented companion article on EVA's four main architectural distinctions

## Suggested reading order

1. `01-paradigm-introduction.md`
2. `02-architectural-contributions.md`
3. `VISUALS/README.md`
4. `THEORY/v0.5-integrated.md`

For implementation status only, see `IMPLEMENTATION/eva-agent-correspondence.md`.
