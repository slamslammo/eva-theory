# Visuals

Reader-facing diagrams and visual aids for `eva-theory`.

This directory holds public-facing visuals that help readers understand EVA's structure and flow without replacing the theory text. The emphasis is on conceptual clarity, not implementation detail or roadmap communication.

## Files in this directory

- `five-layer-overview.html` — static structure view of the five-layer architecture, cross-layer anchors, and peer-circuit action selection
- `signal-flow.html` — flow view showing fast path, slow path, drive broadcast, anchor restriction, and mediated release
- `five-layer-overview-spec.md` — content and reading specification for the five-layer overview visual
- `signal-flow-spec.md` — content and reading specification for the signal-flow visual
- `previews/five-layer-overview-preview.png` — static preview image used in the root README and repository entry page
- `previews/signal-flow-preview.png` — static preview image used in the root README and repository entry page

## Directory boundary

`VISUALS/` is for explanatory diagrams and their supporting assets. It is not for roadmap publication, implementation status diagrams, or source-level technical disclosure.

`THEORY/v0.5-integrated.md` remains the authoritative textual description of the architecture. The visuals here are explanatory companions.
