EVA — Five-Layer Architecture Overview

A public-facing static structure map aligned with v0.5. This overview highlights the strongest current theoretical core (L1–L3), the anchor system as a cross-layer structure, and action selection as a peer circuit rather than a reasoning sub-module.

static structure view cross-layer anchors peer-circuit selection
How to read this diagram. Start with the five layers in the main stack. Then read the anchor spine as a structure that applies across layers rather than inside a single module. Finally, focus on Layer 3: EVA treats the reasoning core and basal ganglia analog as distinct but interacting structures, because candidate generation and action release should not collapse into a single process.

Visual emphasis

  • L1–L3 are the strongest current theoretical core.
  • L4–L5 remain part of the architecture but carry weaker derivational support.
  • Anchors are shown as cross-layer structural constraints, not a rule box.
  • Execution happens only after mediated release.
  • Selection sits beside reasoning, not inside it.

Anchor System

Cross-layer structural constraint. Anchors restrict what the architecture can generate or release; they are not post-hoc rules attached at the end.

constitutional anchors integrity-preserving continuity-bounded sustainability-oriented identity-shaping (later)

The same anchor system applies across reflex, deliberation, and later self-model development.

Downstream extensions
L5

Social Cognition

downstream extension

Social reasoning about conspecifics, coordination, and other-agent modeling. Included for architectural completeness, but dependent on deployment context and currently weaker in derivational support than L1–L3.

L4

Self-Model

emerges from history

Slow-moving self-representation shaped by behavioral history, boundary conditions, and accumulated costs. Present in the architecture, but expected to emerge only after sustained L3 operation.

Strongest current theoretical core
L3

Adaptive Deliberation

strongest current core
Essential function: non-design-time information carriage and use. This is where EVA's strongest structural claims concentrate.

Memory structure

  • salience-weighted episodic memory
  • semantic memory and skill library
  • history shaped by pressure, not neutral recall alone

L3 role

  • uses information not available at initialization
  • supports within-lifetime adaptation
  • integrates sensing, drive context, and memory

Reasoning Core

Working memory current context and symbolic reasoning; the architectural position where an LLM substrate can operate
Value judgment candidate evaluation weighted by current drive state rather than abstract utility alone
Conflict detection detects competing pressures and routes resolution under anchor-bounded constraints
independent peer circuit

Basal Ganglia Analog

  • default inhibition rather than action-readiness
  • selective release of candidate actions
  • reward-prediction-error-based updating
  • habit crystallization alongside deliberate action
Shown beside the reasoning core because EVA treats selection as a parallel architectural function, not a small sub-module inside reasoning. The core does not directly become action.

Execution edge

Environmental action happens only after mediated release. Reasoning produces candidates; the peer circuit gates what is actually executed under default inhibition.

tool use environmental action selected release only
L2

Drive Layer

strongest current core

Internal pressure structure that shapes cognition continuously. Drives are treated as contextual broadcast rather than command, and reflexive response remains available under threat.

Drive structure

  • continuous internal pressure
  • explicitly injected rather than left opaque
  • shared motivational environment for cognition

Reflexive function

  • immediate response under threat
  • parallel to slower deliberation
  • cannot be simply overridden by higher reasoning
L1

Homeostatic Sensing

strongest current core

The architecture's sensing substrate for viability-relevant state. Homeostatic sensing is where the system learns not just what is happening, but what kind of state it is in and whether immediate integrity-preserving response is required.

State sensing

current condition of the agent and its operational environment

Rate sensing

metabolic or burn-rate awareness rather than static state alone

Signal classification

threat / status / background distinction before full interpretation

Caption. High-level structural overview of EVA. The architecture is organized around homeostatic sensing, drive structure, and adaptive deliberation as the strongest current core, with anchors acting across layers and action selection mediated by a peer circuit rather than collapsed into reasoning.
