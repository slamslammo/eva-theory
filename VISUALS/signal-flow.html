EVA — Signal Flow Overview

A public-facing dynamic flow map aligned with v0.5. This diagram emphasizes that EVA is not a single serial pipeline: threats can drive a fast reflex path while slower deliberation proceeds under contextual drive broadcast, anchor-bounded candidate generation, and mediated release.

dynamic flow view drive as context mediated release
How to read this diagram. Start at Sensing, where urgent and non-urgent signals enter together. Then follow the split: an urgent signal can drive a fast reflex path, while slower candidate generation proceeds in parallel. The shaded slow-path zone is important: drive is a contextual broadcast, not a command arrow. Anchors act before candidate generation, and nothing reaches Execution until a separate Selection / release stage opens the gate.

Visual emphasis

  • Fast path handles urgent pressure without waiting for full deliberation.
  • Drive broadcast surrounds the slow path as context rather than issuing a command.
  • Anchor restriction appears before candidate generation, not only at the end.
  • Selection / release is distinct from reasoning and keeps default inhibition in place.

Fast path

Reflexive response handling

  • immediate response priority under threat
  • does not wait for full candidate-space exploration
  • coexists with, rather than replaces, slower deliberation
Urgent pressure can reach release priority quickly while the slower path continues in parallel.
Slow path

Adaptive deliberation under current drive context

Slower but richer processing. The whole zone is embedded in a motivational field rather than being told what to do by a single command source.

continuity pressure resource pressure integrity pressure

Anchor restriction

Constraint enters before or at the boundary of candidate-space formation.

Candidate generation

Produces options within the anchor-bounded space rather than from an unrestricted action set.

Evaluation

Assesses candidates under current drive context, memory, and sensed state.

The slow path should be read as contextual modulation, not instruction following from the drive layer.

Selection / release

default inhibition

Fast-path urgency and slow-path candidates converge here, but release remains mediated by a distinct selection function.

  • reasoning produces candidates, not execution
  • a separate circuit mediates what is actually released
  • default inhibition holds until release occurs
Release gate Opens only when a candidate or reflexive tendency is selected for actual execution.

Execution

Only after mediated release does the architecture cross the edge into environmental action.

tool use environmental action external behavior
  • not sensing → reasoning → action
  • action follows selection, not candidate formation alone
Execution is the endpoint of a gated process, not the automatic continuation of cognition.
Outcome feedback. If included, learning remains secondary in the composition: outcomes can feed back toward the selection system as a reward-prediction or updating signal without changing the main message that flow is parallel, contextual, and gated.
