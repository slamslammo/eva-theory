# Route Selection Notes

This file preserves the existence of the route-selection discussion as part of the publication-facing repository structure.

## Scope

This is a lightweight pointer document rather than a full reconstruction of the longer private discussion history.

It refers to the theory-evolution transition in which EVA moved from earlier framing alternatives toward the integrated `v0.5` position.

## What this discussion was about

The key question was not only how to improve theoretical defensibility, but how to do so **without collapsing EVA from a new paradigm into just another adaptive agent architecture**.

That route-selection process eventually stabilized around:
- preserving `continuous existence first`
- preserving `drive as context`
- preserving `anchor` as structural rather than post-hoc
- preserving the basal-ganglia analog as a peer circuit
- adopting stronger claim separation and epistemic layering without giving up EVA’s paradigm identity

The detailed, full-length discussion history is retained in local/private working materials rather than fully mirrored here.
